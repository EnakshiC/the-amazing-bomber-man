# The Amazing Bomber Man

## Overview

The Amazing Bomber Man is a Bomberman-inspired game developed in Java using libGDX as part of a university software engineering project.

The objective is to navigate a maze, eliminate enemies using bombs, uncover the hidden exit, and escape before the countdown timer expires.

## Features

* Destructible and indestructible walls
* Enemy AI and pathfinding
* Collectible power-ups
* Dynamic HUD
* Main menu and pause functionality
* Victory and game-over screens
* Sound effects and background music
* Adaptive fog-of-war game mode

## My Contributions

My primary contributions focused on gameplay interactions and player experience, including:

* Animation systems
* Gameplay interaction logic
* Fog-of-war implementation
* Menu and game-state transitions
* Player feedback systems

## Fog-of-War Feature

As an extension to the base project requirements, a fog-of-war mode was implemented to limit player visibility and create a more challenging gameplay experience. Different modes include: No Fog, Easy Fog, Medium Fog, Hard Fog, Extreme Fog.

The visible area adapts dynamically to player movement, reducing available information and requiring more strategic navigation through the map.

## Technologies

* Java
* libGDX
* IntelliJ IDEA

## Running the Project

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run the desktop launcher.
4. Load a map and start the game.
